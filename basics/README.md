# Basics of DS-DSA (Concept-First Learning)

This folder explains **only the essential data structures and algorithms**
required for **Data Scientist coding interviews**.

### How to Use This Folder

1. **Start here** before solving problems.
2. Read concept files in `data_structures/` and `algorithms/`
3. Observe **visual explanations** (ASCII diagrams)
4. Run the corresponding Python files in `ds_structures/`
5. Then move to `solutions/`

### What This Folder Is NOT
- ❌ Competitive programming material
- ❌ Advanced graph theory
- ❌ Hard DP
- ❌ Mathematical proofs

### What This Folder IS
- ✅ Intuition
- ✅ Interview-ready reasoning
- ✅ Streaming & analytics focused
- ✅ Data Science relevant

Think of this as your **mental model builder**.
