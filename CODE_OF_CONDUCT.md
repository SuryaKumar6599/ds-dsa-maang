# Code of Conduct

## Our Pledge

We, as contributors and maintainers of **ds-dsa-maang**, pledge to make participation in this project a **harassment-free, inclusive, and professional experience** for everyone, regardless of:

- Experience level
- Gender identity and expression
- Sexual orientation
- Disability
- Personal appearance
- Body size
- Race, ethnicity, nationality
- Age
- Religion or belief system
- Educational background
- Employment status

This project is built **by Data Scientists, for Data Scientists**, and we value respectful collaboration above all.

---

## Our Standards

Examples of behavior that **contribute to a positive environment** include:

- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Giving and receiving constructive feedback gracefully
- Focusing on what is best for the community and the project
- Helping others learn and grow
- Maintaining professionalism in technical discussions

Examples of **unacceptable behavior** include:

- Harassment, discrimination, or personal attacks
- Trolling, insulting or derogatory comments
- Publishing private information without explicit permission
- Unprofessional conduct in issues, pull requests, or discussions
- Demeaning language related to skill level or background
- Spam, self-promotion unrelated to the project, or low-effort noise

---

## Scope

This Code of Conduct applies to **all project spaces**, including but not limited to:

- GitHub repositories
- Issues and pull requests
- Discussions and comments
- Code reviews
- Documentation and examples
- Any other interaction related to this project

---

## Enforcement Responsibilities

Project maintainers are responsible for **clarifying standards**, **enforcing them fairly**, and **taking appropriate corrective action** when necessary.

Maintainers have the right and responsibility to:
- Remove, edit, or reject contributions that violate this Code of Conduct
- Temporarily or permanently ban contributors for repeated or severe violations
- Take action to protect the integrity and safety of the community

---

## Reporting Violations

If you experience or witness unacceptable behavior:

- Please report it **privately** to the project maintainers.
- Provide as much detail as possible (links, screenshots, context).

All reports will be reviewed **confidentially**, and maintainers will respond with appropriate action.
